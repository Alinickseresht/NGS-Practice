# Real-Time PCR (qPCR)

## Purpose
Used for amplification and quantification of DNA/RNA.

## Main Components
- Template DNA
- Primers
- Taq polymerase
- Fluorescent dye/probe

## Key Concept
Fluorescence increases during amplification cycles and is measured in real time.

## Applications
- Gene expression analysis
- Pathogen detection
- Molecular diagnostics
